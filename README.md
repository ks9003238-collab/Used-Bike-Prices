# 🚲 Bike Price Analysis using Data Science

## 🚀 Project Overview

This project focuses on analyzing bike price data to understand key factors affecting pricing. It involves data cleaning, exploratory data analysis (EDA), and visualization to uncover insights from the dataset.

---

## 🎯 Objective

- Analyze bike pricing trends  
- Identify factors influencing bike prices  
- Perform exploratory data analysis (EDA)  
- Generate meaningful business insights  

---

## 📁 Dataset

The dataset contains bike-related information such as:
- Brand / Model  
- Price  
- Engine capacity  
- Mileage  
- Power  
- Other specifications  

---

## ⚙️ Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 🔧 Project Workflow

### 1. Data Preprocessing
- Loaded dataset using Pandas  
- Checked missing values and duplicates  
- Cleaned and formatted data  

---

### 2. Exploratory Data Analysis (EDA)

Performed analysis to understand:

- Price distribution  
- Brand-wise pricing trends  
- Relationship between engine capacity and price  
- Mileage vs price comparison  

---

### 3. Data Visualization

Used charts to visualize insights:

- Bar charts  
- Line plots  
- Scatter plots  
- Heatmaps  

---

## 📈 Key Insights

- Higher engine capacity bikes tend to have higher prices  
- Premium brands dominate the high-price segment  
- Mileage and price show varying relationships depending on bike category  

---

## 🚨 Key Learnings

- Importance of data cleaning in real-world datasets  
- Exploratory Data Analysis helps uncover hidden patterns  
- Visualization improves understanding of data  

---
